Setup

One-time configuration for GitHub, Vercel, and Cloudinary

Step 1: GitHub + Vercel (One-time)

Do this once. After this, hitting "Deploy" in the admin will push your changes and Vercel auto-deploys.

1
Initialize git in your project folder
Open your terminal and run:
# In your website-export/ folder: git init git add -A git commit -m "Initial commit"
2
Create a GitHub repo and push
Install the GitHub CLI if you haven't (brew install gh), then run:
gh repo create digvijay-portfolio --public --push --source=.
Or manually create a repo on github.com and push with the URL it gives you.
3
Connect to Vercel
1. Go to vercel.com → New Project
2. Import your GitHub repo
3. Settings: Framework Preset = Other, Build Command = (leave empty), Output Directory = .
4. Click Deploy

After this, every git push = automatic Vercel deploy. ✅
4
Test it
Go to the Deploy tab and click "Deploy Now". Your Vercel URL will go live in ~30 seconds.
Step 2: Cloudinary Image Uploads (Free)

Cloudinary gives you free image hosting (25 GB/month). This lets you upload new images directly from the admin dashboard.

1
Create a free Cloudinary account
Go to cloudinary.com and sign up for free.
2
Create an unsigned upload preset
1. In Cloudinary dashboard → Settings → Upload
2. Scroll to "Upload presets" → Add upload preset
3. Set Signing Mode = Unsigned
4. Note the preset name (e.g., "ml_default")
3
Enter your Cloudinary details below
Your cloud name is shown on the Cloudinary dashboard homepage.
✅ Saved!

These are saved in your browser only — they're never sent to any server.
